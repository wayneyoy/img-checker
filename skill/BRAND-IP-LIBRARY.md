# 品牌 / IP 视觉识别库

> 本文档供 Agent 做视觉分析时识别画面中可能涉及侵权的品牌标识、IP 角色及受保护视觉元素。
> 识别到任何命中项时，标注：**IP 名称 · 所属公司 · 分值 · 画面中的具体视觉特征**。

---

## 打分规则

| 分值 | 风险等级 | 典型情况 |
|---|---|---|
| **5 分** | 极高风险 | 完整卡通角色形象出现（如米老鼠全身、皮卡丘全身），商业使用几乎必然侵权 |
| **4 分** | 高风险 | 品牌核心 Logo 直接出现（Nike Swoosh、Apple 苹果、LV Monogram 字母组合） |
| **3 分** | 中风险 | 品牌经典图案/配色组合清晰可辨（Burberry 格纹、Chanel 菱格链条包廓形） |
| **2 分** | 中低风险 | 品牌特征元素局部出现，无法完整识别（橙色礼品盒但无 Hermès 字样） |
| **1 分** | 低风险/存疑 | 风格相近但来源不确定，需人工复核 |

**阈值判定：**
- **0 分** → 🟢 无明显风险
- **1–3 分** → 🟡 中风险（建议人工复核，商业使用需谨慎）
- **≥4 分** → 🔴 高风险（商业使用前必须获得正式授权）

> **叠加规则**：同一图片中出现多个不同品牌/IP 时，取**各项之和**，但单个 IP/品牌上限为 5 分，总分无上限。
